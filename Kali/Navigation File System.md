# Navigation File System

---

| ***Command***  |             ***Function***             |
|:--------------:|:--------------------------------------:|
|      pwd       |    Print Current Working Directory     |
|       cd       |            Change Directory            |
| ctrl+l / clear |           Clear the terminal           |
|       ls       |         Display List of Items          |
|     mkdir      |            Make a Directory            |
|     rmdir      |           Remove a Directory           |
|    ls -all     | Display all Files + Hidden + Long List |
